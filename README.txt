My Portfolio
Welcome to my personal portfolio! This project showcases my skills and projects using HTML, CSS, and JavaScript. Created in 2022, it features both dark and light themes and provides an overview of my background and work.

Features
Responsive Design: Adapts seamlessly to different screen sizes.
Dark and Light Themes: Users can switch between themes for better readability and preference.
About Me Section: A comprehensive overview of my background, skills, and interests.
Projects Showcase: A gallery of my projects with descriptions and links.
Contact Form: A functional form for visitors to get in touch with me.
Technologies Used
HTML: Structure of the website.
CSS: Styling and layout.
JavaScript: Interactive elements and theme toggling.
